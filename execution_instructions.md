# Instrucciones de Ejecución del Escenario de Prueba en Apache JMeter

Estas instrucciones te guiarán a través del proceso para ejecutar el escenario de prueba de rendimiento en Apache JMeter. Asegúrate de seguir cada paso detalladamente para obtener resultados precisos y significativos.

## Requisitos Previos

1. **Instalación de Apache JMeter:**
   - Asegúrate de tener Apache JMeter instalado en tu máquina. Si aún no lo has instalado, puedes obtener la última versión desde [el sitio web oficial de Apache JMeter](https://jmeter.apache.org/download_jmeter.cgi).

2. **Clonar este Repositorio:**
   - Clona este repositorio en tu máquina local usando el siguiente comando:
     ```bash
     git clone <URL_DEL_REPOSITORIO>
     ```
   - Cambia al directorio del repositorio recién clonado:
     ```bash
     cd <NOMBRE_DEL_REPOSITORIO>
     ```

## Configuración del Escenario de Prueba

1. **Abre Apache JMeter:**
   - Ejecuta Apache JMeter en tu máquina.

2. **Importa los Scripts de Prueba:**
   - Abre Apache JMeter y carga los scripts de prueba ubicados en la carpeta del escenario que quieres ejecutar .

3. **Ajusta la Configuración del Escenario:**
   - Dentro de Apache JMeter, revisa y ajusta la configuración del escenario según tus necesidades. Puedes configurar la carga, la duración de la prueba, y otros parámetros relevantes en los archivos de script.
   - Debes tener en cuenta también modificar las rutas en las que se almacenarán los archivos de manera local en tu pc 😊


## Ejecución del Escenario de Prueba

1. **Inicia la Ejecución:**
   - Inicia la ejecución del escenario de prueba en Apache JMeter.

2. **Observa el Progreso:**
   - Mientras las pruebas se ejecutan, observa el progreso en tiempo real. Puedes analizar la información presentada en la interfaz de usuario de JMeter.

3. **Revisa los Resultados:**
   - Después de que las pruebas hayan finalizado, revisa los resultados detallados en la carpeta `results`. Los informes generados por JMeter te proporcionarán información valiosa sobre el rendimiento del sistema.
